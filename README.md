# VertexFlow Calculator

Калькулятор спецрозкрою внутрішніх кутів для `.glc` файлів. Працює повністю на стороні клієнта у браузері, підключається як плагін WordPress.

## Встановлення

1. Скопіюйте папку плагіна у `wp-content/plugins/vertexflow-calculator/`.
2. Активуйте плагін у адмін-панелі WordPress або через WP-CLI:

```bash
wp plugin activate vertexflow-calculator
```

## Використання

Додайте шорткод на сторінку або запис:

```
[vertexflow_calculator]
```

Після цього у браузері:
1. Завантажте `.glc` файл (кнопка або drag-and-drop).
2. Натисніть **"Обробити"**.
3. Перегляньте результат у SVG-прев?ю.
4. Натисніть **"Завантажити"** для експорту нового файлу з суфіксом `_special_cut`.

## Технічні примітки

- Всі обчислення виконуються у браузері (без серверних викликів).
- Підтримується кодування `windows-1251` і закінчення рядків `CRLF`.
- Обробляються лише **зовнішні контури** (`outer`). Вирізи (VIREZS) та зони (ZONESLIST) не змінюються.

## Структура проєкту

```
assets/
  css/style.css           # UI стилі
  js/
    app.js                # Точка входу
    components/           # UI компоненти
    logic/                # Обробка геометрії
    utils/                # Парсер/білдер/утиліти
vertexflow-calculator.php
includes/
  class-activator.php
  class-deactivator.php
```
