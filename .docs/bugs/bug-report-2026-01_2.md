## Описание проблемы

Продолжение баг-репорта @.docs\bugs\bug-report-2026-01_1.md

**Краткая формулировка:** Внутренние углы из тестового файла обрабатываются неправильно.

**Подробное описание:**

- Исходные данные: первый контур в тестовом файле имеет вид:

 B_______________________________________________C
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                           E___________________D
 |                           |
 |                           |
 |                           |
 |                           |
 |                           |
 |                           |
 |                           |
 A___________________________F

- Ожидаемое поведение: После обработки с обоих сторон от угла E создаются две дополнительные вершины (точки) на расстоянии 100 мм, их длина увеличивается до 120 мм смещая точку E, все остальные точки контура остаются на своих местах. Плсде этого все точки, начиная от A переименовываются по часовой стрелке таким образом чтобы не нарушался алфавитный порядок именования.

 B_______________________________________________C
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                                               |
 |                         F                     |
 |                            __E________________D
 |                           |
 |                           G
 |                           |
 |                           |
 |                           |
 |                           |
 |                           |
 A___________________________H

- Фактическое поведение: 1. Дополнительные вершины создались у дальнего от обрабатываемого конца одного из сегментов. 2. Точки C и E, F и D, а также отрезки CD, DE, EF наложились друг на друга

 B
 | \   
 |     \            
 |          \                
 |               \             
 |                   \        
 |                     \      
 |                         \  
 |                           CE
 |                           |
 |                          |
 |                         |
 |                        | 
 |                      FD __
 |                           G
 |                           |
 A___________________________H



- Как часто это происходит? Всегда
